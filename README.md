# Portfolio Optimization using Exact and Heuristic Methods

## 📌 Problem
This project solves a portfolio optimization problem by allocating investments across assets to maximize return while minimizing risk.

## ⚙️ Methods Used
- Exact Method: SLSQP (SciPy)
- Heuristic Method: Genetic Algorithm (GA)

## 📊 Data
- Real-world financial data from yfinance
- Assets: AAPL, MSFT, GOOGL, AMZN, TSLA, NFLX, BTC-USD

## 📈 Results
- Exact Method: Higher return, faster computation
- GA: More flexible allocation, slightly lower return

## ⚖️ Comparison
We compare methods based on:
- Return
- Risk
- Runtime

## ▶️ How to Run
1. Open the notebook
2. Run all cells

## 📌 Conclusion
Optimization helps in making better financial decisions by balancing risk and return.
